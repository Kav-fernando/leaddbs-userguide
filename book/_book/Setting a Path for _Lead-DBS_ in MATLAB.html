Setting a Path for Lead-DBS in MATLAB

To add Lead-DBS to the MATLAB directories, do the following:

  1. Go to the File tab.
  2. Click on Set Path....
  3. In the new window click on Add with subfolders....
  4. Browse to the location of the Lead-DBS folder and select it.
  5. Finally click on Saveand Close.

The Lead-DBS folder should now appear in the MATLAB directory.
